# BrainBrainBrain — AI戦略ブレインストーミング

> 考える力を、社内に。

事業課題の分析や新規施策の検討を、AIが多角的にサポートするWebツールです。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![CI](https://github.com/BoxPistols/brain-brain-brain/actions/workflows/ci.yml/badge.svg)

**デモサイト:** https://brain-brain-brain.vercel.app

---

## このツールでできること

「情報は集めたが、どこから手をつけるか決められない」「チームで議論しても同じ視点に偏る」——そのような場面で、AIに壁打ち相手を任せられます。

- 課題を入力すると、AIが状況分析と戦略アイデアを構造的に生成します
- 気になるアイデアを深掘りしたり、方向性を修正して再分析できます
- 分析結果はPDF・PowerPoint・CSVなどにエクスポートして社内共有に使えます
- 分析の深度は4段階。クイックな壁打ちから本格的な戦略検討まで対応します

## 画面の構成と使い方

### 1. 入力画面（左パネル）

分析に必要な情報を入力する画面です。

| 項目 | 説明 |
|------|------|
| プロジェクト名 | 任意。空欄の場合、AIが自動で命名します |
| プロダクト/サービス名 | 分析対象のサービス名を入力します（必須） |
| セッション種別 | 分析の目的を選びます（下記9種類から選択） |
| チーム目標 | 達成したいゴールを入力します（必須） |
| 分析深度 | Quick / Standard / Deep / Expert の4段階 |
| 現状課題 | 解決したい課題を入力します。詳細やサブ課題も追加できます |
| 競合・データ情報 | 競合企業名やKPI実績値を任意で入力できます |

**セッション種別の一覧:**

| 種別 | 内容 |
|------|------|
| サービス・事業改善 | プロダクト戦略、収益性改善 |
| マーケティング・集客 | 認知拡大、流入改善 |
| グロース・収益拡大 | ユニットエコノミクス、解約抑制 |
| 新規事業・イノベーション | PoC検証、PMF達成 |
| 顧客体験・関係構築 | カスタマージャーニー改善 |
| 営業・業務オペレーション | 業務効率化、KPI改善 |
| デジタル化・業務効率化 | DX推進、チェンジマネジメント |
| ブランド・UI統一 | ブランド一貫性、制作効率化 |
| 経営戦略・その他 | 上記に当てはまらないテーマ |

**分析深度の目安:**

| 深度 | 生成数 | 所要時間 | 特徴 |
|------|--------|----------|------|
| Quick | 3件 | 約1分 | 概要レベル。素早い壁打ちに |
| Standard | 6件 | 1〜3分 | 標準的な分析。日常的な検討に |
| Deep | 8件 | 3〜5分 | 詳細分析。フレームワークを使った構造化 |
| Expert | 10件 | 3〜5分 | コンサル級。イシュー再定義・競合戦略分析を含む |

入力が完了したら「戦略アイデア生成」ボタンを押すと、AIによる分析が始まります。

### 2. 分析結果画面（右パネル）

AIが生成した分析結果を確認する画面です。

**状況分析**: 入力内容をもとに、AIが現状を多角的に分析した内容が表示されます。競合情報を入力した場合は、競合分析も含まれます。

**戦略アイデア一覧**: アイデアがカード形式で表示されます。各カードには以下の情報が含まれます。

- タイトルと詳細説明
- 優先度（高/中/低）、工数（大/中/小）、インパクト（大/中/小）
- 実現可能性スコア（Deep以上の深度で表示）

**深掘り**: 画面下部に表示されるサジェスト質問をクリックすると、そのテーマについてAIがさらに詳しく分析します。自由なテキスト入力による深掘りも可能です。

**ドリルダウン**: アイデアカードにカーソルを合わせると表示される矢印ボタンを押すと、そのアイデアを2〜3個のサブプランに分解できます。

**ブラッシュアップ**: 画面下部のテキストエリアに方向性の修正や追加の観点を入力すると、AIが分析内容を改善します。

### 3. 設定画面

画面右上の歯車アイコンから開きます。

**AIプロバイダーの選択:**

| プロバイダー | 説明 |
|--------------|------|
| OpenAI | クラウドAPI。APIキーを入力するとProモードで利用可能 |
| Ollama | ローカルLLM。自分のPCで動作するAI（無料） |
| LM Studio | ローカルLLM。自分のPCで動作するAI（無料） |

**モデル選択**: 「Auto」を選ぶと、タスクに応じて最適なモデルが自動で選ばれます（おすすめ）。手動で特定のモデルを選ぶこともできます。

**接続テスト**: 「接続テスト」ボタンで、AIとの通信が正常かどうか確認できます。

### 4. エクスポート（書き出し）

分析結果は以下の形式でダウンロードできます。

| 形式 | 用途 |
|------|------|
| Markdown (.md) | テキストベースの構造化レポート |
| テキスト (.txt) | シンプルなテキスト形式 |
| CSV (.csv) | Excelなどの表計算ソフトで開けるデータ |
| PDF (.pdf) | 印刷やメール添付用のレポート |
| PowerPoint (.pptx) | 社内プレゼン用のスライド資料 |
| PowerPoint コンサル版 (.pptx) | デザインを強化したプレゼン資料 |

結果画面右上の「プレビュー」ボタンでは、Markdown/テキスト形式のプレビューとソース確認ができます。「ダウンロード」メニューからは全形式を直接ダウンロードできます。

ファイル名にはプロジェクト名と日付が自動で付与されます。

### 5. 履歴管理

画面右上のデータベースアイコンから開きます。

- 過去の分析セッションを一覧で確認できます（最大200件）
- 保存モードを「全Q&A保存」「回答のみ」「保存しない」から選べます
- 履歴はJSONファイルとしてエクスポート・インポートできます

### 6. サポート機能

- **ガイドツアー**: 画面右上のナビゲーションアイコンで、画面の各部分を順番に案内します
- **ヘルプ**: 画面右上の「?」ボタンで、使い方の詳細やショートカットキーを確認できます
- **サポートチャット**: 画面右下の吹き出しアイコンから、よくある質問やAIチャットサポートを利用できます

### キーボードショートカット

| 操作 | ショートカット |
|------|----------------|
| 分析を実行 | Cmd + Enter（Macの場合）/ Ctrl + Enter |
| モーダルを閉じる | Esc |

## データの取り扱いについて

- すべてのデータはお使いのブラウザ内（localStorage）にのみ保存されます
- 外部サーバーやデータベースへの送信はありません
- ただし、AI分析の実行時には入力内容がAIプロバイダー（OpenAIなど）に送信されます
- 企業名の入力は不要です。プロジェクト名で管理できます

## はじめかた

### すぐに試す場合

[デモサイト](https://brain-brain-brain.vercel.app) にアクセスすると、ブラウザだけで利用できます。APIキーなしでも基本機能を試せます。

### APIキーを使う場合

設定画面でOpenAIのAPIキー（`sk-` で始まる文字列）を入力すると、Proモードが有効になります。Expert深度の分析や全モデルの利用が可能です。

APIキーの取得先: https://platform.openai.com

### ローカルLLMを使う場合

OllamaまたはLM Studioをインストールし、設定画面でプロバイダーを切り替えると、自分のPC上でAI分析を実行できます。APIキーは不要です。

## 開発者向け情報

### 技術スタック

| カテゴリ | 技術 |
|---------|------|
| フレームワーク | React 18 + TypeScript |
| ビルドツール | Vite 5 |
| スタイリング | Tailwind CSS 3 |
| テスト | Vitest |
| アイコン | Lucide React |
| PDF出力 | html2pdf.js |
| PPTX出力 | pptxgenjs |

### ローカル開発

```bash
git clone https://github.com/BoxPistols/brain-brain-brain.git
cd brain-brain-brain
npm install
npm run dev
```

http://localhost:5173 で起動します。

### ビルドとプレビュー

```bash
npm run build
npm run preview
```

### テスト

```bash
npm test
```
